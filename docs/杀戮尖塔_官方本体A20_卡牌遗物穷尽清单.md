> **用途**：为状态向量 s 的卡牌/遗物维度提供权威穷尽值。仅包含官方本体、普通模式（Ascension 20），不含 Mod、自定义模式、Blight 遗物。
>
> **数据来源**：Slay the Spire Wiki (slaythespire.wiki.gg)、SlayTheSpire.gg、社区数据、Mod Raw_Data 验证。
>
> **验证日期**：2025-02

## 一、卡牌穷尽清单

### 1.1 总览

| 类别 | 数量 | 说明 |
|------|------|------|
| 铁甲战士 (Ironclad) | 73 | 含 Basic / Common / Uncommon / Rare |
| 静默猎人 (Silent) | 73 | 含 Basic / Common / Uncommon / Rare |
| 缺陷机器人 (Defect) | 73 | 含 Basic / Common / Uncommon / Rare |
| 观者 (Watcher) | 75 | 含 Basic / Common / Uncommon / Rare |
| 无色 (Colorless) | 47 | 商店/事件获得，含 Special |
| 诅咒 (Curse) | 14 | 可进入牌库的诅咒 |
| 状态 (Status) | 12 | 战斗中可进入牌库的状态牌 |
| **合计（去重后用于向量）** | **~270** | 实际编码时需合并 Strike/Defend 等跨职业同名 |

**说明**：Mod 发送的卡牌 `id` 格式通常为 `Strike_G`（静默）、`Strike_R`（铁甲）、`Strike_B`（缺陷）、`Strike_P`（观者）；无色卡无后缀。编码器需做 ID 归一化（大小写、空格/下划线互换）。

**职业卡池例外**：绝大部分情况下，猎人（静默）只能获得本职业 + 无色 + 诅咒 + 状态牌。**唯一例外**：当猎人拿到遗物 **棱镜 (Prismatic Shard)** 后，战斗奖励中会出现其他职业（铁甲、缺陷、观者）的卡牌。因此，若要做「世界第一猎人 AI」且覆盖棱镜局，卡牌向量必须支持全职业 ~270 维，而不能只做静默 146 维。

## 三、与 encoder_ids.yaml 的对应关系

| 项目 | 当前 encoder_ids | 穷尽清单 | 建议 |
|------|---------------------|----------|------|
| 卡牌 | ~200 条，部分重复/缺漏 | 270 | 按穷尽清单重排，补全观者/无色，去重 |
| 遗物 | ~180 条，有重复 | 179 | 去重，统一 ID 格式（如 Snake Skull → Snecko Skull） |
| 药水 | ~40 | 约 40 | 已基本覆盖 |
| Power | ~60 | 随游戏机制 | 保持 |
